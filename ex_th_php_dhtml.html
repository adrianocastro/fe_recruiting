The picture and text is drawn from a database that stores the entire profile information. Only a few of those profile fields are used in this module.

Build this functionality using cross-browser client-side code (for IE5+/NS7.1+), PHP, and a MySQL database. Optimize for download & render speed and server performance. You may use fake images for the purposes of this exercise.
