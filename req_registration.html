Frontend Engineer for Registration & Core Services

We're looking for an experienced, self-motivated, and passionate frontend engineer to join the Platform Engineering team. You'll work closely with interaction & visual designers, user experience researchers, backend engineers, marketers, and product managers on the integral Core Services product group, the team that is responsible for building the functionality that integrates the entire Yahoo! network together.

The ideal candidate will have a strong command of web standards, CSS-driven/tableless HTML layouts, DHTML (JavaScript 1.0-1.5, CSS1 & CSS2, DOM Levels 0-2), PHP, Unix, HTTP (get/post, headers), Apache, cross-browser compatibility & browser degradation strategies, optimization techniques (both in file weight and perceived/actual rendering speed), object-oriented design, unit testing & debugging techniques, dynamic content retrieval, state management, accessibility techniques, device-portability techniques, internationalization issues, and version control software. A Computer Science degree or related work experience is also required.

The environment is fast-paced with numerous user-centered iterative design & development cycles. You'll be responsible for building everything from usability prototypes to deployment-quality code. In Core Services, you'll help build user interfaces that every Yahoo! user will use and see. There is also the opportunity to build innovative new UI widgets that can be included in our code libraries and used across the Yahoo! network!

If you're interested, please provide URLs and code samples of your work. Having your own site is a bonus!
