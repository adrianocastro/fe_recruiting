Q: What color is each sentence in this webpage, assuming it renders correctly? 

	<html>
	<head>
	<style type="text/css">
	p { color: blue; }
	div#one p { color: green; }
	div#one>p { color: red; }
	</style>
	</head>
	<body>
	<p>Sentence 1.</p>
	<p>Sentence 2.</p>
	<div id="one">
		<p>Sentence 3.</p>
		<p>Sentence 4.</p>
		<div>
			<p>Sentence 5.</p>
			<p>Sentence 6.</p>
			</div>
		<p>Sentence 7.</p>
	</div>
	</body>
	</html>
A: 1 and 2 are blue; 3, 4 and 7 are red, and 5 and 6 are green.) (Bonus question: Netscape 7 and IE6, which one gets this page right and renders it correctly? Answer: N7 gets it right, IE6 renders the red sentences as green.)
