Phone Screen Questions

The purpose of the phone screen is to determine if the candidate is worth bringing in for a short 1st round interview. This is not meant to be a thorough technical assessment, only a high-level assessment of the candidate's skills and interest in the position. Therefore, these questions are merely high-level guidelines. Specific questions tailored to the candidate's previous work experience can also be used. The levels listed below can help gauge the candidate's skill rating.

  • General
    • All levels - For which browsers do you typically develop?
  • CSS
    • All levels - In what kinds of ways have you used CSS before?
    • All levels - What are the pros and cons of using CSS?
    • Level 1 - Are you familiar with the display (or float) property? Tell me about it.
    • Level 2 - Are you familiar with the concept of the box model? Tell me about it.
    • Level 3 - Are you familiar with the various doctypes that can be used on a page? Tell me about them.
  • JS
    • All levels - What kinds of functionality have you written in JavaScript?
    • All levels - What are the pros and cons of using JavaScript?
    • Level 1 - What are some ways in which you've done form validation?
    • Level 2 - Are you familiar with the document object model? Tell me about it. How does it differ on different browsers?
    • Level 3 - Are you familiar with the various JavaScript event models? Tell me about them.
  • PHP
    • All levels - What kinds of functionality have you written in PHP?
    • All levels - What are the pros and cons of doing form validation with PHP, as opposed to JavaScript?
    • Level 1 - Are you familiar with the global variables automatically provided to you from an HTTP request? Tell me about all of them.
    • Level 2 - What are all the ways in which you can maintain a user's state in PHP?
    • Level 3 - Are you familiar with register_globals? Tell me about it. What are other configurations in the php.ini file that you would modify for a live site?
